# Versão original em Portugol

Esta pasta contém a versão original do projeto feita em Portugol durante uma atividade do curso.

Essa versão foi mantida apenas como referência histórica e para mostrar a evolução da ideia até a adaptação em C.

A versão principal do projeto está na implementação em C.

## Sobre esta versão

Na versão em Portugol, optei por não utilizar arrays e matrizes, porque o trabalho era em dupla e isso poderia dificultar o entendimento da minha dupla naquele momento.

Por esse motivo, a estrutura do código foi feita de uma forma mais simples e direta. Também utilizei algumas abreviações em nomes de variáveis para reduzir a quantidade de escrita e facilitar o desenvolvimento durante a atividade.

Mesmo não sendo a versão mais organizada, ela teve importância no processo, porque serviu como base para a criação da versão em C.

Na versão em C, comecei a adaptar a lógica original usando arrays, matrizes, structs e funções de forma mais estruturada.

## Como executar a versão em Portugol

Para executar esta versão, é necessário utilizar o site Portugol Dev.

Acesse o site `portugol.dev`, abra o editor, cole ou carregue o arquivo `.por` e execute o código por lá.

Esta versão foi mantida apenas como referência. A versão principal do projeto está sendo desenvolvida em C.
