# Projetos em C

Repositório com projetos e exercícios feitos durante meus estudos de programação em C.

A ideia principal é praticar lógica, organização de código e conceitos básicos da linguagem, registrando minha evolução com o tempo.

## Projetos

Aqui estão projetos simples de estudo, como sistemas de cadastro, gerenciamento de tarefas, cálculos e exercícios acadêmicos.

Alguns códigos estão mais organizados que outros, pois fazem parte do meu processo de aprendizado.

## Observação

Todos os códigos deste repositório têm pontos que podem ser melhorados.

Alguns projetos talvez não sejam mais continuados exatamente como estão, pois foram feitos para praticar conceitos específicos e me aperfeiçoar naquele momento.

Mesmo assim, mantenho eles aqui como registro da minha evolução e continuo melhorando minha forma de programar conforme aprendo novos conceitos.
